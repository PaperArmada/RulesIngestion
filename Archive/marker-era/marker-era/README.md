# Marker-era archive

This folder contains documents from the **Marker-first ingestion model** (RulesIngestion Mark II). They are kept for **provenance only** and are **not normative for Mark III**.

**Do not reason from these documents** when making design or implementation decisions. Mark III assumes a prose-first, DeepSeek-centric pipeline; the primary unit of ingestion is authored text (prose), not marker blocks or chunks.

Every document in this folder has a one-line banner at the top:

> This document reflects a Marker-first ingestion model and is not normative for Mark III.

Canonical Mark III design lives in `Docs/Design/`:

- `RULES_INGESTION_MARK_III_OVERVIEW.md`
- `STAGE_A_PROSE_RECONSTRUCTION.md`
- `STAGE_B_EVIDENCE_BINDING.md`
